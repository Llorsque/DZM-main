# De Zilveren Bal 2026 — Hoofdtoernooi

Interactieve startlijst tool voor De Zilveren Bal Hoofdtoernooi 2026.

## Gebruik

Open `index.html` in een webbrowser — geen installatie nodig.

## Functies

### Dames / Heren
Schakel bovenin de header tussen de dames- en herencompetitie.

### Presentatie modus
- Toont de huidige en volgende rit met deelnemersinformatie
- Navigeren: **pijltjestoetsen ← →** of knoppen in de header
- Klik op een rit in de linkerkolom om er direct naartoe te springen

### Startlijst invullen
1. Klik op **STARTLIJST INVULLEN** in de header
2. Kies per rit (Wit/Rood/Geel) welke deelnemer in welke baan rijdt
3. Voeg ritten toe met **+ RIT TOEVOEGEN**
4. Start Omloop 2 met de knop onderaan
5. Klik **OPSLAAN & SLUITEN** — de startlijst wordt lokaal opgeslagen

> De startlijst wordt bewaard in de browser (localStorage). Openen op een andere computer begint leeg.

## Banen
| Kleur | Baan |
|-------|------|
| ⚪ Wit | Binnen |
| 🔴 Rood | Midden |
| 🟡 Geel | Buiten |

## Deelnemers
- **24 dames** + 2 reserves
- **25 heren** + 3 reserves
