# De Zilveren Bal 2026 — Hoofdtoernooi

Startlijst tool voor het hoofdtoernooi van De Zilveren Bal 2026.

## Gebruik
Open `index.html` in een browser, of host via GitHub Pages.

## Programma invullen
Klik "PROGRAMMA INVULLEN" om ritten toe te voegen en rijders in te vullen.
Alle data wordt opgeslagen in localStorage.
